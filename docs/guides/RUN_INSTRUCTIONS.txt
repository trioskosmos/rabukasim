Open a NEW Terminal tab/window, navigate to the folder again, and run:
--------------------------------------------------------------------------------
cd 'C:\Users\trios\.gemini\antigravity\vscode\loveca-copy'
$env:PYTHONPATH="."; uv run ai/train_optimized.py


--------------------------------------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------------------------------------
- If it crashes with "missing DLL", re-install vc_redist.x64.exe.
- If it says "CUDA not available", check your NVIDIA drivers.
- To stop training: Press Ctrl+C in the training terminal. It will save a checkpoint.
