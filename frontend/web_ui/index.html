ヘルプ & FAQ (Help)

デッキのセット方法 (Deck Setup)

  • Deck Log: Bushiroad Deck Log からHTMLをコピーして「Option 2」に貼り付けます。
  • Test Deck: 「Load Test Deck」ボタンで、テスト用（2倍量）デッキを読み込みます。
  • Random Deck: 「Generate Random Deck」ボタンで、ランダムな有効デッキを生成します。

📱 モバイル操作 (Mobile)

  • 画面右下の Menu ボタン でサイドバー（アクション履歴、ログ）を開閉できます。
  • 手札やステージなどのゾーンは横スクロール可能です。

🎮 ゲーム操作 (Controls)

  • 更新 (Update): ゲームの状態を再取得します。
  • リセット (Reset): ゲームを初期状態の最初からやり直します。
  • カードをクリックして使用・移動します。
  • アクションを選択したら、サイドバーの「Do Action」等は自動で進む場合があります。
